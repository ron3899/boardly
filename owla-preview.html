Roadmap Feature

New Roadmap Page

This feature adds a dedicated roadmap section to Boardly, giving teams visibility into what's being built and what's coming next.

Key Features:

  • Three-column timeline view (Now, Next, Later)
  • Color-coded status indicators
  • Hover effects and smooth animations
  • Consistent dark theme styling
  • Responsive design for all screen sizes

The roadmap is accessible from the main navigation sidebar and automatically highlights when active.
