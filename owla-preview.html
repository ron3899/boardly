This demo showcases all the dark mode improvements made to the Boardly web application. Every component has been audited and fixed to ensure proper contrast, visibility, and consistency in both light and dark modes.

  • Kanban view - Columns, cards, backgrounds, and borders

  • All UI components - Dropdown menus, dialogs, sheets, and selects
